USAGE GUIDE

Installation

Installation: git clone https://github.com/Bajinder17/NetShark
pip install -r requirements.txt

Basic Commands

# Basic capture on default interface
python network_analyzer.py

# Specify interface
python network_analyzer.py -i eth0

# Apply BPF filter
python network_analyzer.py -f "tcp port 80"

# Enable verbose output
python network_analyzer.py -v

# Custom output file
python network_analyzer.py -o custom_capture.pcap

Key Features

  • Automatic interface detection if not specified
  • Live statistics display with Ctrl+C to stop
  • Automatic PCAP file generation
  • Memory-efficient packet processing
  • Colorized output for better readability
